# Changelog

All notable changes to the VectorDB Studio extension will be documented in this file.

## [0.2.1] - 2024

### Added
- Connection management for Milvus instances
- Database creation and deletion
- Collection management (create, load, release, drop)
- Data viewing with pagination
- Data insertion and deletion
- Partition support
- Schema viewer for collections
- Configurable page size and connection timeout
